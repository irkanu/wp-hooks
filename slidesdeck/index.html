Definitions

  • Hooks - General term referring to places where you add code to WordPress core, a theme or a plugin.
  • Filters - Let you to get and modify data.
  • Actions - Allow you to run code at a specific point in the WordPress runtime.

All the Hooks in WordPress

  • $wp_filters - Associative array of all actions and filters
